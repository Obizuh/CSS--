CSS(层叠样式表)

层叠样式表(Cascading Style Sheets，缩写为CSS)，是一种 样式表语言，用来述 HTML 或 XML(包括 SVGMathML、XHTML 类的 ML 分支语言) 文档的呈现.CSS 述了在屏幕、纸质、音频等其它媒体上的元景应该如何被渲染的问题。
CSS是开放网的核心语言之一，由 W3C 规范 实现跨测觉器的标准化。SS 节省了大量的工作样式可以通过定义保存在外部,CS 文件中，同时控制多网页的布局，这意着开发者不必经历在所有网页上编布局的麻烦，SS 被分为不同等级:SS1 现已废弃，(SS21 是推荐标准，(SS3 分成多个小模块目正在标准化中。
